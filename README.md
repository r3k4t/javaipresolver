Javaipresolver is a java program for find out any website ip address.

### Setup ###

+ sudo apt install openjdk-8-jdk openjdk-8-jre

### Terminal Command ###

+ git clone https://github.com/r3k4t/javaipresolver.git
+ cd javaipresolver
+ sudo java javaipresolver.java
